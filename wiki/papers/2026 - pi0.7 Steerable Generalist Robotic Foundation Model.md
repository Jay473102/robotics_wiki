# $\pi_{0.7}$: a Steerable Generalist Robotic Foundation Model with Emergent Capabilities

## TL;DR

$\pi_{0.7}$은 diverse robot data, autonomous/failure data, egocentric human data, multimodal web data를 rich context conditioning으로 묶어 학습한 steerable generalist VLA다. 핵심은 language instruction뿐 아니라 subgoal images, episode metadata, quality/length 같은 정보를 prompt에 넣어 mixed-quality data를 활용하고, out-of-the-box dexterity, cross-embodiment transfer, compositional generalization을 얻었다는 주장이다.

## Problem

기존 VLA는 많은 task를 학습해도 unseen task나 unseen embodiment에서 skill을 새 방식으로 조합하는 compositional generalization이 제한적이다. 또한 mixed-quality data를 단순히 합치면 서로 다른 strategy와 quality가 평균화되어 성능이 떨어질 수 있다.

## Core Idea

$\pi_{0.7}$은 "무엇을 할지"뿐 아니라 "어떻게 할지"를 context로 제공한다. Prompt에는 detailed language labels, subtask instructions, generated subgoal images, episode metadata, strategy/quality/length metadata 등이 포함된다.

이 conditioning은 diverse data에서 mode averaging을 줄이고, failure/suboptimal/autonomous/non-robot data도 의미 있는 training signal로 쓰게 한다.

## Method

논문은 $\pi_{0.7}$을 5B-parameter VLA로 설명한다. 구성은 4B VLM backbone, MEM-style video history encoder, 약 860M action expert를 포함한다. Action expert는 flow matching 또는 diffusion 계열 objective로 action chunk를 생성하고, VLM backbone은 FAST token supervision과 knowledge insulation recipe를 사용한다.

Subgoal image는 high-level policy와 world model을 통해 생성될 수 있고, inference에서는 language coaching이나 generated subgoal image를 통해 새 task를 가르치는 방식도 제시한다.

## Data / Embodiment

학습 데이터는 robot demonstrations, autonomous data, multimodal web data, egocentric human data를 포함한다고 설명한다. Evaluation에는 여러 robot platform이 포함되며, cross-embodiment 실험에는 bimanual/static robot과 UR5e 계열 platform 등이 등장한다.

## Evaluation

평가 축은 다음과 같다.

- Out-of-the-box dexterity: espresso, laundry, box building 등.
- Broad instruction following and referential instructions.
- Dataset-bias breaking through instructions.
- Zero-shot cross-embodiment transfer.
- Language coaching and generated subgoal image guidance.
- Scaling/ablation: metadata, diverse data, non-robot data의 기여.

## Results

- 논문은 $\pi_{0.7}$이 espresso machine operation, laundry folding, trash bag removal, box folding, vegetable peeling 같은 dexterous task를 task-specific post-training 없이 수행한다고 주장한다.
- Cross-embodiment에서는 laundry folding을 본 적 없는 target robot으로 전이하는 사례를 제시한다.
- Air fryer sweet potato loading처럼 action-level data가 없는 task도 step-by-step language coaching과 generated subgoal images로 수행하는 사례를 제시한다.
- Ablation은 metadata/context가 mixed-quality data와 diverse data 활용에 중요하다고 보고한다.

## Limitations

- Zero-shot generalization success rate는 in-distribution task보다 낮다고 논문도 인정한다.
- 성능은 private data scale, prompt metadata 품질, generated subgoal image 품질에 크게 의존할 수 있다.
- 논문은 많은 qualitative/figure 중심 결과를 포함하므로, 독립 재현 가능한 benchmark와 open model/data 여부는 별도 확인이 필요하다.
- 검증 필요: 실제 모델/데이터 공개 범위, deployment safety policy, compute/data 규모의 세부 수치.

## Open Questions

- Rich prompt metadata 없이도 $\pi_{0.7}$의 성능이 유지되는 task와 무너지는 task는 어떻게 구분되는가?
- Subgoal image/world model이 틀린 목표를 제시할 때 low-level policy가 이를 검출하거나 무시할 수 있는가?
- Cross-embodiment transfer를 위해 action space, proprioception, embodiment metadata를 어떻게 표준화해야 하는가?
